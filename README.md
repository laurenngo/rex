# rex

a [Sails](http://sailsjs.org) application

##WHAT IS REX?

REX is a mobile-first app for people to share their travel recommendations with their friends through email. 

##TECHNOLOGIES USED:

- AngularJS
- Node.js
- Sails.js
- MongoDB
- SendGrid
- Materialize UI
- Angular Material UI

##ADDITIONAL FEATURES TO WORK ON:
